# G.todo

## Todo App with React

> Developing time: 2021.10.30 - 2021.11.14

## Features

- Basic todos:

  - List all todos
  - Add a todo
  - Delete a todo
  - Update a todo
  - Mark a todo as done
  - Show done/todo ratio

- Additionals
  - Change theme: warm / chill
  - Show navbar when scroll down
  - Scroll up button on the navbar

## Libraries

- React Icons
  - https://react-icons.github.io/react-icons/
